# TravelTrucks

Web application for camper rentals in Ukraine.

## Description

TravelTrucks is a platform where users can browse and book campers for travel. The application allows filtering vehicles by various criteria, viewing detailed information, and making reservations.

## Main Features

- Browse catalog of available campers
- Filter by location, vehicle type, and equipment
- Add campers to favorites
- View detailed information about each camper
- Read reviews from other users
- Book campers through a convenient form
- Load more results with pagination

## Technologies

- Next.js 15
- TypeScript
- Zustand (state management)
- Axios (API requests)
- CSS Modules
- React Hook Form + Zod (form validation)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/OlenaZhmudenko/travel-trucks.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```

4. Open in browser: http://localhost:3000

## Project Structure

- `/app` - pages and layouts
- `/components` - React components
- `/lib` - API, stores, utilities
- `/public` - images and icons
